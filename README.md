# KSystemStats Scripts

This is a fork of [https://github.com/KerJoe/ksystemstats_scripts](https://github.com/KerJoe/ksystemstats_scripts), refer to this for cofiguration.

This repository exists as a convenience for creating and publishing the package on COPR, all credits go to the orginal author.

## Installation

```bash
sudo dnf copr enable itotm/ksystemstats-scripts
sudo dnf in ksystemstats-scripts
```
